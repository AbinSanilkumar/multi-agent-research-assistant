# Multi-Agent Research Assistant

An AI-powered research platform that uses multiple specialized agents to plan, research, verify, and generate professional reports.

## Features

- Research Topic Submission
- Multi-Agent Workflow
- Fact Verification
- Report Generation
- RAG Integration (Upcoming)
- User Authentication

## Tech Stack

- Django
- LangChain
- LangGraph
- Gemini/OpenAI
- PostgreSQL
- FAISS
